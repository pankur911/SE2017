Code Samurai at School Of Computer Science

This project is about creating a website for the Code Samurai. Mentee can login to make an appointment with Mentor for specific class. The website will display courses, student name and Mentor information. It will provide detail report in the end of semester about Mentee information; what time Mentee Sign In and Sign out; Which Mentor helped for which course for how long. Also at end of the day it will calculate how many hours each individual Mentor helped for any specific course for how many total hours. It will accumulate for whole semester and at the end of semester gives the entire report with total hours and which course most students coming for tutoring.
